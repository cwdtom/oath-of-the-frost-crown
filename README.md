# Oath of the Frost-Crown

A Godot 4.7 2D action game prototype.

## Requirements

- Godot 4.7 or newer

## Running

1. Open this folder in Godot.
2. Open `project.godot`.
3. Run the main scene.

## Controls

- Left / Right: move
- Space: jump
- A: attack

## Project Structure

- `main.tscn` - main scene
- `player/` - player scene, script, and animation setup
- `assets/player/` - player sprite frames and Godot import metadata
