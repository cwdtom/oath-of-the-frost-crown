# Keep Enemy skills species-owned

The shared Enemy lifecycle treats Player presence in a Skill Detection Area as a persistent trigger, but it does not auto-discover or orchestrate arbitrary skill nodes. Boss variants continue the existing Bear King and Wolf King specialization pattern: the Elk King reuses Elk thunder and shield behavior while owning independent thunder and earthquake lifecycles, allowing concurrent casts and preserving casts that began before defeat without introducing a generic multi-skill framework.
